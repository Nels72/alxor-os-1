Installer Alex sur votre appareil (optionnel, recommandé)

Alex fonctionne comme une application sans passer par l'App Store ou le Play Store.

iPhone / iPad

  1. Ouvrez le lien ci-dessus dans Safari
  2. Appuyez sur l'icône Partager (carré avec flèche vers le haut)
  3. Sélectionnez "Sur l'écran d'accueil"
  4. Confirmez → l'icône Alex apparaît sur votre écran

Android

  1. Ouvrez le lien dans Chrome
  2. Un bandeau "Installer Alex" apparaît en bas de l'écran
  3. Appuyez sur Installer → c'est fait

PC / Mac

  1. Ouvrez le lien dans Chrome ou Edge
  2. Cliquez sur l'icône d'installation dans la barre d'adresse (ou le bandeau "Installer")
  3. Alex s'ouvre dans sa propre fenêtre, comme une application
